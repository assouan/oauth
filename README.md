# Assouan OAuth

OAuth helpers and authorization-code client for the A PHP libraries.

```bash
composer require assouan/oauth
```

Requires PHP 8.5 or later.

## Classes

- `A\OAuth\AuthorizationCode`
- `A\OAuth\AuthorizationRequest`
- `A\OAuth\Client`
- `A\OAuth\OAuthException`
- `A\OAuth\Pkce`
- `A\OAuth\Token`
